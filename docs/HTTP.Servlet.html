CLASS HTTP.Servlet @abstract @public

summary

     

Base class for all Servlets.

$addPath(path) @public  Add a path that this servlet will serve for.
$handle(context) bool @public  Check if this servlet wants to serve the req.
$setMimeType(type) @public  Set the MIME type to be used in the response for this Servlet.
$serve(context) bool @public  Serve the req, put the reply in resp. Return TRUE when the request was handled.
 
Known subclasses:
     
HTTP.FileServlet
HTTP.FunctionServlet
HTTP.RedirectServlet
HTTP.RpcServlet
HTTP.StringServlet
 

members (alphabetically)

     

PROC $addPath(string path) @public

     

Add a path that this servlet will serve for.

FUNC $handle(HTTP.Context context) bool @public

     

Check if this servlet wants to serve the req.

If so, put the result in resp and return TRUE. If not return FALSE.

FUNC $serve(HTTP.Context context) bool @abstract @public

     

Serve the req, put the reply in resp. Return TRUE when the request was handled.

This is invoked from the handle() method. Servlet implementations usually only need to implement this method.

PROC $setMimeType(string type) @public

     

Set the MIME type to be used in the response for this Servlet.
