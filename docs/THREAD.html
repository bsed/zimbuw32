FUNC eval<Titem>(func< => Titem> f) pipe<Titem> @public

     

Create a thread to evaluate f and return a pipe from which the result can be read.

Example: 

 pipe<string> p = THREAD.eval({ => "hello there" })
 IO.print(p.read())
