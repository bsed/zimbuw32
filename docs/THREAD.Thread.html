CLASS T.thread @public

summary

     

The builtin type thread.

A thread that will execute a method.

Example: run myProc() in a thread: 

 thread t = NEW(myProc).start()
 t.wait()

Example: subclass 

 CLASS MyThread EXTENDS thread
  PROC $body() @replace
    int count
    WHILE TRUE
      IO.print(count)
      ++count
      TIME.sleep(500'000)
    }
  }
 }

 MyThread t = NEW(doSomething)
 t.start()
 TIME.sleepSec(3)
 t.kill()
