CLASS HTTP.RpcServlet @public

summary

     

A Servlet that handles RPC requests.

Requests normally come in at "/rpc?ServiceName.Method".

NEW() @public  Create a new RpcServlet.
$init() @public 
$addHandler(rpcInfo, handler) @public  Set the handler function to call for RPC call rpc
$serve(context) bool @public  Handle a request: Check that it starts with "rpc?name=ServiceName.method" Check that the service name matches "ServiceName" Find a handler for "method" and call it.
